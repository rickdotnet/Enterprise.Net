# Enterprise.Net

Enterprise.Net is a .NET library for creating hierarchical, system-oriented architectures. It aims to simplify the development of complex systems by providing a structured approach to task management and message routing. It's an evolution of the [Apollo](https://github.com/rickdotnet/apollo) library.
